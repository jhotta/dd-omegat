Datadog-Riak Integration

Overview

Connect Riak to Datadog in order to:

  • Visualize Riak performance and utilization.
  • Correlate the performance of Riak with the rest of your applications.

For more details about configuring this integration refer to the following file(s) on GitHub:

  • Riak YAML example
  • Riak checks.d
