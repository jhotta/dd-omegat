APIライブラリー

公式ライブラリー

Datadog APIを操作するために、Datadogが公式に公開しているライブラリーです。

Python

  • DogApi - Python Datadog API ラッパー.
  • dogstatsd-python - Python DogStatsD クライアント.

Ruby

  • DogApi - Ruby Datadog API ラッパー.
  • dogstatsd-ruby - Ruby DogStatsD クライアント.

PHP

  • php-datadogstatsd - PHP DogStatsD クライアント.

C#

  • dogstatsd-csharp-client C# DogStatsD クライアント.


コミュニティーライブラリー

Datadog APIを操作するために、コミュニティーのメンバーが開発/公開しているライブラリーです。

Java

  • java-dogstatsd-client - Jave向けDogStatsDクライエント by Indeed.
  • metrics-datadog - codahale’のmetricsライブラーリのバックエンド by Coursera.
  • Lassie - Jave向け screenboard APIクライエント by Bazaarvoice.

Node.js

  • node-datadog - Node.js向け APIクライエント by HashGo.
  • node-dogstatsd - Node.js向け DogStatsDクライエント by Young Han Lee.
  • node-dogapi - Node.js向け APIクライエント by Brett Langdon.

Perl

  • webservice-datadog -　Perl向け APIクライエント by Jennifer Pinkham.

  • dogstatsd-perl - a Perl DogStatsD client, contributed by Stefan Goethals.
  • dogstatsd-perl - Perl向け DogStatsDクライエント by Stefan Goethals.

Ruby

  • metricks-dogstatsd - 有名なMetriks　gemのバックエンド by Mavenlink.

PHP

  • plesk_metrics_datadog - Pleskからメトリクスを収集するためのPHPのスクリプト by Israel Viana.

Go

  • go-datadog-api - Go向け APIラッパーwrapper　by Mark Smith.
  • go-dogstatsd - Goで開発されたdogstatsdクライエント by Ooyala.

Python

  • scales_datadog - ScalesライブラリーのDatadogバックエンド by Tommaso Barbugli.


インテグレーションライブラリー

Saltstack

  • Datadog Salstack Formula
  • Datadog Saltstack by Luca Cipriani

Ansible

  • Ansible script - Datadogのイベントストリームにイベントを投稿するためのコールバックプラグイン by Alek Storm.

FreeSwitch

  • FreeSwitch-DataDog-Metrics - FreeSwitch ESLアプリケーションの統計情報をDogStatsD APIを使ってDatadogに書きすための仕組みです。 by WiMacTel.

Google Analytics

  • casperjs-google-analytics-realtime-scrape - Datadog APIを使って、Google Analyticsからデータを取り込むためのライブラリー.

Pid-stats

  • pid-stats - StatsDからプロセス情報を生成するライブラリー by GitterHQ.

Logstash

  • dogstatsd.rb - Logstashからデータを取り込むためのライブラリー by DatadogメンバーのConor Branagan.

Reverse Color Scheme

  • デフォルトより暗い背景色を設定するためのに、サイトの色味を変更する CSSライブラリー.
